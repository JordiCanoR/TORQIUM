Sistema IA Predictivo

Control y mantenimiento inteligente de 8 motores · PLC Siemens · IA · Flask · Snap7

Resumen del proyecto

Este sistema integra un PLC Siemens (como tu S7-1200) con un servidor en Python (Flask) para controlar y supervisar 8 motores eléctricos. La solución combina automatización clásica con inteligencia artificial para anticipar fallos, optimizar mantenimientos y mejorar la seguridad de la instalación.

  • Control individual y general de 8 motores.
  • Sensores de temperatura dedicados por motor.
  • Mantenimiento manual y mantenimiento inteligente por IA.
  • Reglas configurables por el usuario y reglas generadas automáticamente por IA.
  • Paradas de emergencia generales y por motor.
  • Registro diario automático en CSV.
  • Alarmas y avisos constantes (fallos, mantenimientos cumplidos o por cumplir).
  • Control por voz y por pantalla táctil.
  • Supervisión y avisos gestionados por algoritmos de IA.

Arquitectura del sistema

La arquitectura se basa en la comunicación entre el PLC Siemens y un servidor en Python mediante la librería Snap7. Este servidor expone una API REST que permite al frontend web consultar estados, enviar órdenes y recibir predicciones de la IA.

PLC Siemens ⇄ Snap7 ⇄ Servidor Flask (API REST) ⇄ Modelos IA ⇄ Frontend web

El sistema registra los datos de funcionamiento de los motores, analiza el historial con modelos de machine learning y genera predicciones de mantenimiento, además de disparar alarmas cuando detecta comportamientos anómalos.

Funcionalidades clave

Control de 8 motores

Encendido, apagado y modos de funcionamiento por motor, con posibilidad de automatizar reglas de arranque y parada según horarios, temperaturas o condiciones de proceso.

Mantenimiento predictivo

Los datos de uso y temperatura se registran y procesan mediante modelos de IA para estimar cuándo conviene realizar el mantenimiento antes de que aparezcan fallos críticos.

Alarmas avanzadas

El sistema genera alarmas por sobretemperatura, tiempos excesivos de funcionamiento, paradas inesperadas o incumplimiento de mantenimientos programados, con avisos en pantalla y por voz.

Registro y trazabilidad

Cada día se genera un CSV con el estado de los motores, incidencias, temperaturas y acciones realizadas. Esto permite llevar un histórico detallado para auditorías o análisis posteriores.

Interfaz de usuario

El panel de control web permite visualizar el estado de los 8 motores, consultar temperaturas, ver alertas activas y revisar las recomendaciones de la IA. Además, el control por voz simplifica la operación en entornos industriales donde el operador no siempre puede usar teclado o ratón.

La interfaz está pensada para ser clara, rápida y usable en pantallas táctiles industriales.

Valor para la industria

Este proyecto demuestra cómo la combinación de PLCs clásicos con inteligencia artificial y servicios web puede transformar el mantenimiento industrial, pasando de un modelo reactivo (reparar cuando se rompe) a un modelo predictivo e inteligente que reduce paradas, mejora la seguridad y optimiza recursos.
